A mobile app to quickly create Spotify playlists. You can swipe right on a song you like to add to playlist. Built with PhoneGap.

First Person Mechanic: Major League Hacking First Place Award

A virtual reality game using Oculus Rift and Microsoft Kinect. You can use your own hands via Kinect to assemble/disassemble a car engine and bicycle. Built with Unity3D.

Created by Josue Cruz, Shariq Hashme, and Yoonshik Hong in April 2014

A home security system. When an intruder enters the house, or when there is a fire in the house, it sends you a notification. Built with an Arduino.

Created by Jeffrey He, Yoonshik Hong, Andrew Liu, and Craig Weiss in March 2014
